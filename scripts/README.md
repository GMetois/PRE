## Custom Python packages management

### Install

To install the custom Python packages:

1. Make the install script executable:

```console
$ chmod +x install
```

2. Run the script:

```console
$ ./install
```

### Uninstall

To uninstall these packages:

1. Make the uninstall script executable:

```console
$ chmod +x uninstall
```

2. Run the script:

```console
$ ./uninstall
```
